## 📤 Uploading Your Videos to GitHub

To embed videos directly in README, upload them to GitHub:

### Method 1: Via GitHub Issues (Easiest)
1. Go to any GitHub repository
2. Open a **New Issue**
3. **Drag and drop** your `.mp4` file into the text area
4. Copy the generated link (looks like: `https://github.com/user-attachments/assets/...`)
5. Paste the link directly in your README
6. Close the issue (you don't need to submit it)

### Method 2: Via Repository Assets
1. Go to your repository **Settings**
2. Navigate to **Assets** section
3. Upload your video files
4. Copy the asset URLs

### Method 3: Direct Repository Upload
1. Upload `.mp4` files to your repository
2. Use raw URL in markdown
